# Advice Ticket
![Screenshot 1](/Screenshot%20(28).png)
## Description

The "Advice Ticket" is a web application that provides users with random pieces of advice. It generates a unique advice number and a QR code for each piece of advice, making it a fun and interactive way to discover helpful insights.

## Features

- Fetches random advice from the [Advice Slip API](https://api.adviceslip.com/).
- Displays the advice along with an advice number.
- Offers a responsive and visually appealing user interface.

## Demo

You can see a live demo of the "Advice Ticket" [here](http://pcoder.me/AdviceTicket/).

## Installation

1. Clone this repository:

    ```bash
    git clone https://github.com/Pratyush-Nirwan/AdviceTicket    
    ```
2. Navigate to the project directory:

    ```bash
    cd advice-ticket
    ```
3. Open the index.html file in your web browser.

### Usage

1. Open the webpage in your browser.
2. Click the GitHub icon to explore more.
3. Enjoy the random pieces of advice and share them using the generated QR codes.
